薬剤師とファーマコメトリクス

Chapter 1 薬物動態学

書籍「第4版 臨床薬物動態学」を参考文献として、薬物動態およびPKPD関連のあれこれをR言語とともに考えてみる。対象薬はPKPD分野で馴染み深いレボフロキサシンとした。

1.1 対象薬：レボフロキサシン

1.1.1 薬物動態パラメータ

薬物動態パラメータは、臨床薬物動態情報局のデータを利用させていただきました。

F <- 0.98  # バイオアベイラビリティ
Ae <- 0.939  # 尿中未変化体薬物排泄率
CLtot <- 161.5  # クリアランス（mL/min）
Vd <- 111.1  # 分布容積（L）
fuP <- 0.72  # 血漿（血清）中薬物非結合形分率
BP <- 0.8  # B/P 全血液中薬物濃度/血漿（血清）中薬物濃度比

Fが0.98と非常に高い。内服と点滴静注製剤で同一用量であることも納得である。
Aeも0.939と非常に高値。腎排泄型薬剤と判断できる。
fuPは0.72（> 0.2）であり、血漿たん白結合の変動に影響をほぼ受けない（binding insensitive）と考えられる。

1.1.2 分布容積

上記の分布容積Vdは血漿（血清）中薬物濃度を基にしたものであるので、全血液中薬物濃度から計算した分布容積（Vd_aとする）を知りたい。そこで、B/Pを用いて計算する。参考文献 p.28あたりを確認。

Vd_a <- Vd / BP
Vd_a
## [1] 138.875

Vd_aが大きい（> 50）ので、レボフロキサシンは、そのほとんどが細胞内液中に存在すると考えられる。

1.1.3 クリアランス

# 腎クリアランス
CLr <- CLtot * Ae
CLr
## [1] 151.6485
# 腎外クリアランス（肝クリアランス）
CLer <- CLtot - CLr
CLh <- CLer
CLh
## [1] 9.8515

上記より、体内からの排出は、ほとんど腎で行われていることが分かる。 やはり、レボフロキサシンの場合には腎外クリアランスを無視してよさそうだ。

1.1.4 腎抽出比

レボフロキサシンが腎臓で排出される際、腎臓にinした分のうち、どれくらいの割合が清浄化されるのかを知りたい。
分母として血漿（血清）と血球を合わせた「腎全血流速度（QR）」を使用するので、分子の「清浄化の速度（CLr）」も全血のそれを使用しなければいけない。そこで、分布容積と同じく、B/Pにより補正する。参考文献 p.29あたりを参照。

QR <- 1200  # 腎全血流速度
Er <- (CLr / BP) / QR  # 腎抽出比
Er
## [1] 0.1579672

腎抽出比は十分に小さい（< 0.3）ので、腎クリアランスは、腎臓の清浄化能力が律速となっている（消失能依存型）と判断する。

1.1.5 薬物動態的特徴

ここまでの確認で、レボフロキサシンの以下の薬物動態的特徴が分かった。

  • 腎排泄型薬剤である。
    • さらに、消失能依存型といえる。
  • 分布容積は大きめ。

これらの特徴により、レボフロキサシンは、参考文献のp.57 表A4・12のタイプBに該当すると考えられる。

1.1.6 定常状態血中濃度

\[CB_{ssave(po)} = (Fa \cdot D / τ) / (fuB \cdot CLintR)\]

Fa <- 1
D <- 500 * 1000
tau <- 24 * 60
CLintR <- 60
fuB <- fuP / BP

CBssavef <- (Fa * D / tau) / (fuB * CLintR)
CBssavef  # 単位は、µg/mL
## [1] 6.430041
